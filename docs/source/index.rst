Welcome to PEARLM's documentation!
==================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

**PEARLM** is a Python framework about Knowledge Graph and explaination-based language model.
It is also the code associated with paper "PERLM: Faithful Path-Based Explainable Recommendation via Language
Modeling over Knowledge Graphs".

.. note::
   This project is under development.
